# Lazy Load Vue.js Components When They Become Visible

[![Patreon](https://img.shields.io/badge/patreon-donate-blue.svg)](https://www.patreon.com/maoberlehner)
[![Donate](https://img.shields.io/badge/Donate-PayPal-blue.svg)](https://paypal.me/maoberlehner)

This is an example project for the following article: [Lazy Load Vue.js Components When They Become Visible](https://markus.oberlehner.net/blog/lazy-load-vue-components-when-they-become-visible/)

## Build Setup

```bash
# Install dependencies.
npm install

# Serve with hot reload.
npm run serve

# Build for production with minification.
npm run build
```
